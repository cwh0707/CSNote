# 二叉树的遍历

### 二叉树的定义

```c++
//树节点的定义
struct TreeNode{
  int val;
  TreeNode* left;
  TreeNode* right;
  TreeNode(int x): val(x), left(NULL), right(NULL){}
};
```

```python
class TreeNode:
	def __init__(self, x):
    self.val = x
    self.left = None
    self.right = None
```

> 二叉树的遍历主要分为两种：递归写法和迭代写法。其中迭代写法就是模拟一个栈
